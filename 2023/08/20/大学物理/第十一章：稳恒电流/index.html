稳恒电流

电流强度：

I=dqdtI = \frac{ {\rm d}q}{ {\rm d}t} I=dtdq​

电流密度矢量：

j=dIdSj = \frac{ {\rm d}I}{ {\rm d}S} j=dSdI​

方向是该点正电荷的运动方向

漂移运动：在电场的作用下，载流子作相应于电场的定向运动

漂移速度：漂移运动的平均速度叫做漂移速度

事实上，对于电流密度矢量和漂移速度，有如下的关系：

<p></p>j=−nevd<p></p> \pmb{j} = -ne\pmb{v_d} <p></p>j=−nevd​

有电荷守恒定律的数学表达式：

∮SjdS=−dqdt\oint_S\pmb{j}{\rm d}\pmb{S} = -\frac{ {\rm d}q} { {\rm d}t} ∮S​jdS=−dtdq​

稳恒电流意味着：

∮SjdS=0\oint_S\pmb{j}{\rm d}\pmb{S} = 0 ∮S​jdS=0

欧姆定律

欧姆定律的基本形式：

I=URI = \frac{U}{R} I=RU​

欧姆定律的微分形式：

j=γEj = \gamma E j=γE

其中 γ=1ρ\gamma = \frac{1}{\rho}γ=ρ1​ 是电导率

焦耳定律的微分形式：

ω=γE2\omega = \gamma E^2 ω=γE2

电动势

电动势：在电源内部，把单位正电荷从负极移动到正极时非静电力所做的功

E=AkqE = \frac{A_k}{q} E=qAk​​

或者说还有下面的式子：电动势是单位正电荷绕闭合回路一圈时非静电力所做的功

E=∮Ek⋅dlE = \oint\pmb E_k\cdot {\rm d}\pmb{l} E=∮Ek​⋅dl
