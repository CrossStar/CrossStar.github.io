LaplaceLaplaceLaplace 变换的各种性质

微分性质

针对象原函数的

L[f′(t)]=sF(s)−f(0)L[f^{\prime}(t)] = sF(s) - f(0) L[f′(t)]=sF(s)−f(0)

针对象函数的

L−1[F′(s)]=−tf(t)L^{-1}[F^{\prime}(s)] = -tf(t) L−1[F′(s)]=−tf(t)

事实上，上面的式子还有如下变形:

L[(−t)nf(t)]=F(n)(s)L−1[F(s)]=−1tL−1[F′(s)]\begin{align} L[(-t)^n f(t)] = F^{(n)}(s)\\ L^{-1}[F(s)] = -\frac{1}{t}L^{-1}[F^{\prime}(s)] \end{align} L[(−t)nf(t)]=F(n)(s)L−1[F(s)]=−t1​L−1[F′(s)]​​

其中:

  • (1) 式用于求解正向的变换，用来分离象原函数中 ttt 的限制
  • (2) 式用于求解逆向的变换，有的时候先对象函数求导 （比如含有 ln⁡f(s)\ln f(s)lnf(s) 类的式子）然后再进行逆向变换会方便很多

积分性质

针对象原函数的

L[∫0tf(T)dT]=1sF(s)L[\int_0^t f(T) {\rm d} T] = \frac{1}{s} F(s) L[∫0t​f(T)dT]=s1​F(s)

针对象函数的

L[f(t)t]=∫s∞F(s)dtL[\frac{f(t)}{t}] = \int_s^\infty F(s) {\rm d} t L[tf(t)​]=∫s∞​F(s)dt

时移性质

L[f(t−t0)]=e−st0F(s)L[f(t - t_0)] = e^{-st_0}F(s) L[f(t−t0​)]=e−st0​F(s)

或者:

L−1[e−st0F(s)]=f(t−t0)L^{-1}[e^{-st_0}F(s)] = f(t - t_0) L−1[e−st0​F(s)]=f(t−t0​)

上面的式子常常用于逆向变换，用于把象函数中的 e−st0e^{-st_0}e−st0​ 分离

频移性质

L[es0tf(t)]=F(s−s0)L[e^{s_0t}f(t)] = F(s - s_0) L[es0​tf(t)]=F(s−s0​)

或者：

L−1[F(s−s0)]=es0tf(t)L^{-1}[F(s - s_0)] = e^{s_0t}f(t) L−1[F(s−s0​)]=es0​tf(t)

记住：

  • ttt 代表时间，sss 代表频率
  • este^{st}est 作用在象函数上时，是时移性质，时间移动一个常数；作用在象原函数上时，是频移性质，频率移动一个常数

L[sinh⁡t]=ωs2−ω2L[\sinh t] = \frac{\omega}{s^2 - \omega^2} L[sinht]=s2−ω2ω​
