样本空间和随机空间

样本空间和随机事件

必然现象与随机现象.

随机试验：
（1）可重复
（2）每次结果不确定，但所有结果确定

  • 样本空间：随机实验的所有可能结果构成的集合，记为 SSS

  • 样本点：试验的每一个结果，或者说样本空间 SSS 中的每一个元素

  • 随机事件：样本空间的任意一个子集

    • 基本事件：只含有一个样本的子集

  • 必然事件：在任何一次试验中一定会发生的事件

  • 不可能事件：在任何一次试验都不会发生的事件

事件的相互关系和运算

  • 包含：如果事件 BBB 发生的时候，一定有 AAA 发生，那么就称事件 AAA 包含事件 BBB ，记作 A⊃BA \supset BA⊃B
    • 相等：如果同时有 A⊃BA \supset BA⊃B 以及 B⊃AB \supset AB⊃A, 那么称AAA和BBB相等.记作A=BA = BA=B
  • 和事件：A∪BA\cup BA∪B 当且仅当 AAA 或 BBB 发生
  • 积事件：A∩BA\cap BA∩B 当且仅当 AAA 且 BBB 发生

我们把：

⋃j=1nAj⋂j=1nAj\bigcup_{j = 1}^n A_j \quad \bigcap_{j = 1}^nA_j j=1⋃n​Aj​j=1⋂n​Aj​

  • 逆事件：对于“AAA不发生”叫做AAA的逆事件，记作A‾\overline{A}A.
    • 对立事件：AAA和A‾\overline{A}A互为逆事件
  • 差事件：A−B={x:x∈A  and  x∉B}A - B = \{x:x\in A \; and\; x\notin B\}A−B={x:x∈Aandx∈/B}
    • A−B=AB‾A - B = A\overline{B}A−B=AB

条件概率

定义1.4.1: 如果P(B)>0P(B) > 0P(B)>0, 那么在BBB发生的条件下AAA发生的概率叫做条件概率：

P(A∣B)=P(AB)P(B)P(A|B) = \frac{P(AB)}{P(B)} P(A∣B)=P(B)P(AB)​

条件概率是在新样本空间中的概率度量

乘法公式

事实上，由条件概率的定义，我们有：

P(AB)=P(A)P(B∣A)=P(B)P(A∣B)P(AB) = P(A)P(B|A) = P(B)P(A|B) P(AB)=P(A)P(B∣A)=P(B)P(A∣B)

这是概率的乘法公式.

事件的独立性与独立试验

定义1.5.1：对A,BA, BA,B两个事件，当 P(AB)=P(A)P(B)P(AB) = P(A)P(B)P(AB)=P(A)P(B) 成立时，称A,BA,BA,B相互独立

定义1.5.2: 对A,B,CA, B, CA,B,C三个事件，

  • 当 P(AB)=P(A)P(B)P(AB) = P(A)P(B)P(AB)=P(A)P(B)，P(BC)=P(B)P(C)P(BC) = P(B)P(C)P(BC)=P(B)P(C), P(AC)=P(A)P(C)P(AC) = P(A)P(C)P(AC)=P(A)P(C) 成立，称A,B,CA,B,CA,B,C两两独立.
  • 如果A,B,CA, B, CA,B,C两两独立，并且还有P(ABC)=P(A)P(B)P(C)P(ABC) = P(A)P(B)P(C)P(ABC)=P(A)P(B)P(C), 称A,B,CA,B,CA,B,C相互独立.
