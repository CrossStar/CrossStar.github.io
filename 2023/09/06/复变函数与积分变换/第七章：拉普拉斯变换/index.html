  2. 求 LTLTLT 的解析性时，可以把积分次序和微分次序调换

几个重要的 LTLTLT 变换

L[sin⁡ωt]=ωs2+ω2L[\sin \omega t] = \frac{\omega}{s^2 + \omega^2} L[sinωt]=s2+ω2ω​

L[cos⁡ωt]=ss2+ω2L[\cos \omega t] = \frac{s}{s^2 + \omega^2} L[cosωt]=s2+ω2s​

L[sinh⁡t]=ωs2−ω2L[\sinh t] = \frac{\omega}{s^2 - \omega^2} L[sinht]=s2−ω2ω​

L[cosh⁡t]=ss2−ω2L[\cosh t] = \frac{s}{s^2 - \omega^2} L[cosht]=s2−ω2s​

L[e−λtsin⁡ωt]=ω(s+λ)2+ω2L[e^{-\lambda t}\sin \omega t] = \frac{\omega}{(s + \lambda)^2 + \omega^2} \\ L[e−λtsinωt]=(s+λ)2+ω2ω​

L[e−λtcos⁡ωt]=s+λ(s+λ)2+ω2L[e^{-\lambda t}\cos \omega t] = \frac{s + \lambda}{(s + \lambda)^2 + \omega^2} L[e−λtcosωt]=(s+λ)2+ω2s+λ​

L[tα]=Γ(α+1)sα+1(α>−1,Re(s)>0)L[t^\alpha] = \frac{\Gamma(\alpha + 1)}{s^{\alpha + 1}}\quad(\alpha > -1, {\rm Re}(s) > 0) L[tα]=sα+1Γ(α+1)​(α>−1,Re(s)>0)

当 aaa 是自然数时，有：

L[tα]=α!sα+1L[t^\alpha] = \frac{\alpha!}{s^{\alpha + 1}} L[tα]=sα+1α!​

∫0+∞f(t)tdt=∫0+∞F(s)ds其中,L[f(t)]=F(s)\int_0^{+\infty}\frac{f(t)}{t}{\rm d}t = \int_0^{+\infty}F(s){\rm ds}\\ \text{其中}, L[f(t)] = F(s) ∫0+∞​tf(t)​dt=∫0+∞​F(s)ds其中,L[f(t)]=F(s)
