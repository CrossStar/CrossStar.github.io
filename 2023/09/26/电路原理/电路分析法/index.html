电路分析法

本文最后更新于：2023年9月26日 中午

主体有 3 种方法：

支路电流法

所有支路的电流都要求，电路比较简单的时候可用。按照结点和回路分开列电路方程

对于含有电流源的支路，一般把这个支路作为连支（也就是在列 KVLKVLKVL 的时候避开电流源支路）

回路电流法

在 回路数 < 节点数 的时候可用

着重考虑电流源

默认：网孔回路法

着重考虑电流源：

  • 选择单连支回路作为回路，理想电流源支路不能选为树支
  • 遇到含有电流源的支路的时候，恰当选择单连支回路让电流源在回路的最外面（也就是 电流源电流 = 回路电流）
  • 如果仍然存在理想电流源，引入电流源的电压作为变量，然后补充方程

列写方程组的时候，只需要考虑那些回路中不含电流源的回路

Ri1Il1+Ri2Il2+⋯+RiiIli+⋯+R1nIln=∑USR_{i1}I_{l 1} + R_{i2}I_{l 2} + \cdots + R_{ii}I_{li} + \cdots + R_{1n}I_{l n} = \sum U_{S} Ri1​Il1​+Ri2​Il2​+⋯+Rii​Ili​+⋯+R1n​Iln​=∑US​

其中，

  • Rij(i≠j)R_{i j}(i \neq j)Rij​(i=j) 为互电阻，是 iii 回路和 jjj 回路的共同电阻之和（电流绕向相同为正，绕向相反为负）
  • RiiR_{ii}Rii​ 为自电阻，是 iii 回路上的电阻之和
  • ∑US\sum U_{S}∑US​ 表示这个回路中的所有的电压源电压值之和，关联参考方向为负

自电阻压降 ±\pm± 互电阻压降 = ∑\sum∑ 电压源

本质上仍然是 KVLKVLKVL，但不同的地方在于 KVLKVLKVL 是以电阻为中心，回路法是以回路的电压为中心

节点电压法

在 节点数 < 回路数 的时候可用

着重考虑电压源：

  1. 如果支路中存在理想电压源，那么一般把理想电压源两端中的一端选作参考节点
  2. 如果选定之后，电路中仍然存在理想电压源，那么把理想电压源先当作电流源，然后根据电压关系补充方程
  3. 考虑使用广义节点，把理想电压源用广义节点包裹，然后补充方程

Gi1U1+Gi2U2+⋯+GiiUi+⋯+GinUn=∑iGUS+∑iISG_{i1}U_{1} + G_{i2}U_{2} + \cdots + G_{ii}U_{i} + \cdots + G_{in}U_{n} = \sum_{i} GU_{S} + \sum_{i} I_{S} Gi1​U1​+Gi2​U2​+⋯+Gii​Ui​+⋯+Gin​Un​=i∑​GUS​+i∑​IS​

其中，

  • Gij(i≠j)G_{ij}(i \neq j)Gij​(i=j) 为互电导，是 iii 节点和 jjj 节点 两节点之间各支路的电导之和 （电流源支路除外）总取负值
  • GiiG_{ii}Gii​ 是自电导，是连接 iii 节点各支路的电导之和 （电流源支路除外），总取正值
  • ∑iGUS\sum_{i} GU_{S}∑i​GUS​ 是连接 iii 节点各支路上的电压源的电压值（“+” 对着节点为正 ；“-” 对着节点为负）和这条支路上的电导乘积之和
  • ∑iIS\sum_{i}I_{S}∑i​IS​ 是连接 iii 节点各支路上电流源的电流值之和（流入为正，流出为负）

自电导电流 - 互电导电流 = ∑\sum∑ 电压源电流 + ∑\sum∑ 电流源

本质上仍然是 KCLKCLKCL ，和 KCLKCLKCL 的不同在于 KCLKCLKCL 注重观察支路的电流，而节点电压法注重对节点处的电流分类
