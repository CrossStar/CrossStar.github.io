静电场中的金属导体

补充：常用形体的场强分布

圆柱面的场强分布：

在圆柱面外，有

E=λ2πε0rE = \frac{\lambda}{2\pi \varepsilon_0 r} E=2πε0​rλ​

其中，λ\lambdaλ 是圆柱体表面电荷的线密度

在圆柱面内，有

E=0E = 0 E=0

无限大均匀平板的场强分布：

E=σ2ε0E = \frac{\sigma}{2\varepsilon_0} E=2ε0​σ​

其中，σ\sigmaσ 是无限大均匀平板电荷的面密度

导体的静电平衡

导体的静电平衡状态是指：当导体处在电场中时，导体内自由电子的宏观定向运动完全停止的状态

导体处于静电平衡状态时，具有如下三条性质：

  • 导体内部场强处处为零
  • 导体表面外侧，紧靠导体处的电场和导体表面垂直
  • 导体是个等势体，导体表面是个等势面

导体处于静电平衡状态时的电荷分布

  • 当导体内无空腔时，导体内处处无净电荷（电荷分布在导体表面）

​ 当导体内有空腔时，除开无空腔的部分无净电荷之外，空腔表面净电荷必须为零，空腔内的场强也必是零

注意对空腔的理解：空腔指的是一个内部不含任何导电性物质的区域

  • 导体表面的场强和导体表面的电荷面密度成正比，或者说：

E=σε0E = \frac{\sigma}{\varepsilon_0} E=ε0​σ​

  • 导体表面某处的曲率越大，该处的电荷面密度越大

导体接地

导体接地存在两条性质：

  • 接地导体的电势为0
  • 如果导体本身存在电荷并且外部没有其他电荷，那么接地导体上的电荷会被中和

​ 如果导体本身存在电荷并且外部存在其他电荷，那么接地导体上的电荷由外部电荷决定，带电量由电势的两种计算方式列方程组得出（P89 T10.4 & T10.5）

电容 电容器

孤立导体的电容

孤立导体的电容被定义为：

C=QUC = \frac{Q}{U} C=UQ​

其中 UUU 是导体的电势

电容的物理意义：使导体升高单位电势所必须给予的电量

电容器的电容

电容器：我们使用两个带电的相邻导体来储存电荷和电量，我们把这样的装置叫做电容器

电容器的电容被定义为：

C=QUA−UBC = \frac{Q}{U_A - U_B} C=UA​−UB​Q​

常用电容器的电容

平行板电容器的电容：

C=ε0SdC = \varepsilon_0\frac{S}{d} C=ε0​dS​

圆柱形电容器的电容：

C=2πε0lln⁡RBRAC = \frac{2\pi \varepsilon_0 l}{\ln \frac{R_B}{R_A}} C=lnRA​RB​​2πε0​l​

球形电容器的电容：

C=4πε0RARBRB−RAC = \frac{4\pi \varepsilon_0 R_AR_B}{R_B - R_A} C=RB​−RA​4πε0​RA​RB​​

电容器的串联和并联

电容器顺次连接叫做电容器的串联，总电容满足如下公式：

1C=1C1+1C2+⋯+1Cn\frac{1}{C} = \frac{1}{C_1} + \frac{1}{C_2} + \cdots + \frac{1}{C_n} C1​=C1​1​+C2​1​+⋯+Cn​1​

电容器头对头，尾对尾的连接方式叫做电容器的并联，总电容满足如下公式：

C=C1+C2+⋯+CnC = C_1 + C_2 + \cdots +C_n C=C1​+C2​+⋯+Cn​

静电场中的电介质

相对介电常数：电容器充满某种电介质时的电容 CrC_rCr​ 和电容器间真空时的电容 C0C_0C0​ 之比叫做这种电介质的相对介电常数，记作 ε0\varepsilon_0ε0​

ε0=CrC0\varepsilon_0 = \frac{C_r}{C_0} ε0​=C0​Cr​​

平行板间的场强：

E=E0εrE = \frac{E_0}{\varepsilon_r} E=εr​E0​​

电介质的极化

电矩：在电场力的作用下，无极分子的正、负电荷中心在外电场方向上产生相对位移 lll，电矩 p=ql\pmb{p} = q\pmb{l}p=ql ，方向由负电荷指向正电荷

电介质的极化：在外加电场中，电介质表面出现极化电荷的现象叫做电介质的极化

位移极化：无极分子电介质体内的电荷在外加电场的作用下发生了位移

取向极化：有极分子电介质体内的电荷在外加电场的作用下发生了取向的改变

极化强度：引入体积元 ΔV\Delta VΔV，在宏观上足够小而在微观上足够大，电介质在某点处的极化强度 P\pmb{P}P 由如下式定义：

P=∑pΔV\pmb{P} = \frac{\sum \pmb{p}}{\Delta V} P=ΔV∑p​

对于线性电介质而言，极化强度又满足如下的关系：

P=ε0χeE\pmb{P} = \varepsilon_0 \chi_e \pmb{E} P=ε0​χe​E

其中，χe\chi_eχe​ 是一个无量纲的常数，叫做电极化率；E\pmb{E}E 是场源电荷和极化电荷的合场强

均匀电介质极化时，电介质表面上某点的极化电荷面密度 σ′\sigma^\primeσ′ 等于极化强度在该点处的法向分量

电介质中静电场的基本定理

对于各向同性均匀电介质，有：

1+χe=εr1 + \chi_e = \varepsilon_r 1+χe​=εr​

E=E0εrE = \frac{E_0}{\varepsilon_r} E=εr​E0​​

σ′=σ0(1−1εr)\sigma^\prime = \sigma_0(1 - \frac{1}{\varepsilon_r}) σ′=σ0​(1−εr​1​)

电位移：

D=ε0E+P\pmb{D} = \varepsilon_0\pmb{E} + \pmb{P} D=ε0​E+P

借助电位移，我们有介质中的高斯定理：

∮DdS=∑q\oint\pmb{D}{\rm d}\pmb{S} = \sum q ∮DdS=∑q

需要注意，上式中的 qqq 是自由电荷，而非极化电荷

和无介质高斯定理的联系：

无介质高斯定理的形式为：

∮EdS=∑qε0(*)\oint E{\rm d}S = \frac{\sum q}{\varepsilon_0} \tag{*} ∮EdS=ε0​∑q​(*)

这个式子需要 在对电荷求和之后除以 ε0\varepsilon_0ε0​

当不存在介质时，εr=1\varepsilon_r = 1εr​=1，此时电位移矢量 D=ε0E\pmb{D} = \varepsilon_0\pmb{E}D=ε0​E. 于是就同样有：

∮DdS=∑q\oint\pmb{D}{\rm d}\pmb{S} = {\sum q} ∮DdS=∑q

对于各向同性的电介质，有：

D=εE\pmb{D} = \varepsilon \pmb{E} D=εE

ε\varepsilonε 是电介质的介电常数，ε=ε0εr\varepsilon = \varepsilon_0 \varepsilon_rε=ε0​εr​

处于静电平衡的导体是一个等势体

带电导体总的电势升高，因为带电导体移向无穷远处所需要克服的电场力做功增大（不带电导体和带电导体间存在引力）

小球接地之后：球壳和小球之间存在电场，小球电势为零

10.4:\color{red}{10.4}:10.4:

在金属球壳内，作出包围两个带电导体的高斯面。高斯面包围的净电荷为 e>0e > 0e>0 ，这意味着电场线会穿出高斯面到达电势为 UUU 的球壳。从球壳到高斯面，电势一直升高，而从高斯面到电荷为 2e2e2e 的导体表面，电势再次升高，也就证明了电荷为 2e2e2e 的导体的电势大于 UUU
