复数的定义

复数是形如 $z = x + iy;(x, y\in R)$的数

  • 一般情况下，两个复数不能比较大小

复平面与复数的模及辐角

  • 复数 $z = x + iy$ 与复平面上的点 $P(x, y)$ 一一对应
  • 复数 $z = x + iy$ 也可用从原点指向 $P$ 的向量 $\overrightarrow{OP}$ 来表达

事实上，有

$$
|Rez| \leq |z| \leq |Rez| + |Imz| \
|Imz| \leq |z| \leq |Rez| + |Imz| \
$$

当 $z \neq 0 $ 时，复数表示的向量和实轴正向的夹角称为 $z$ 的辐角，记作

$$
\theta = Arg,z
$$

这意味着复数的辐角有无数多个，我们把满足 $-\pi < \theta \leq \pi$ 的辐角 $\theta_0$ 称为辐角$Arg , z$的主值，也就是辐角主值。有如下记法：

$$\theta_0 = arg ,z$$

一般而言，辐角主值$\theta_0$和辐角$\theta$满足如下关系：

$$ \theta = \theta_0 + 2k\pi \quad k\in Z$$

也可以写成：

$$Arg,z = arg,z + 2k\pi \quad k \in Z$$

复数的三角形式和指数形式

  • 复数的三角形式是： $z = r(\cos\theta + i\sin\theta)$
  • 复数的指数形式是： $z = re^{i\theta}$

其中 $r = |z|$

我们有棣莫佛公式：
$$
z^n = r^n(\cos n\theta + \sin n\theta) = r^n e^{i,n\theta}
$$

复数的运算

复数共有如下几种运算

  • 加法运算
  • 减法运算
  • 乘法运算
  • 除法运算
  • 取模运算
  • 共轭运算

在复数域内，我们熟知的运算律和代数恒等式仍然成立

关于取模运算和共轭运算，补充下面几个式子：

复数模的三角不等式：

$$
|z_1 + z_2| \leq |z_1| + |z_2| \
|z_1 - z_2| \geq ||z_1| + |z_2||
$$

共轭复数在复平面内是关于实轴对称的，这意味着

$$
|z| = |\overline{z}|
$$

如果 $z$不在负实轴和原点上还应该有

$$
arg,z = -arg,\overline{z}
$$

复数乘法$z = z_1z_2$的意义是，构造一个新的复数$z$，其模是$|z_1||z_2|$,辐角是$z_1, z_2$两个辐角之和。换句话说，也就是：

$$
|z_1z_2| = |z_1||z_2| \
Arg(z_1z_2) = Arg,z_1 + Arg,z_2
$$

除法是乘法的逆运算，因此有

$$
|\frac{z_1}{z_2}| = \frac{|z_1|}{|z_2|} \
Arg(\frac {z_1}{z_2}) = Arg,z_1 - Arg,z_2
$$

复数的乘幂和方根

乘幂主要为棣莫佛公式
下面我们着重讨论复数的方根运算

复数$n$次方根的几何意义为$n$等分单位圆
观察到复数的指数形式易于进行方根运算，因此我们需要如下步骤

  1. 把复数化为其指数形式
  • 这里指数形式需要考虑其辐角，而不是辐角主值
  1. 对复数的指数形式进行方根运算
  2. 把复数的指数形式化为原来的形式

例：现有 $z = 1+ i$ ，求 $\sqrt[4]{z}$.
解：
$z$ 的模 $|z| = \sqrt{2}$，辐角主值 $\theta_0 = \frac{\pi}{4}$
因此$z = \sqrt{2}e^{\frac{\pi}{4} + 2k\pi}\quad(k \in Z)$，$\sqrt[4]{z} = \sqrt{2}e^{\frac{\pi}{16} + \frac{k\pi}{2}}\quad(k = 0, 1, 2, 3)$

复球面与无穷远点

复球面：用来引入无穷远点

连接球面的北极点和$P$点，直线交$C$平面于一个点，那么久可以将复平面上的点和球面上的点建立起一一对应关系。

球面的北极点对应无穷远点，球面的南极点对应原点。

注：扩充复平面上的$\infty$点只有一个点，而不分$+\infty$与$-\infty$点

复平面上的点集

对复平面上的点集，我们有如下七个概念：

(1)邻域

  • $D(z_0,\delta) = {z;|z - z_0| < \delta}$
  • $D(z_0, \delta) \backslash {z_0}$ 为去心邻域

无穷远点的邻域：

$${z;|z| > R}$$

无穷远点的去心邻域：

$${z;R < |z| < +\infty}$$

(2)内点，开集

(3)边界点、边界
$\partial E$ 或 $bd(E)$

(4)区域

(5)有界区域

(6)简单曲线、光滑曲线

曲线：
由方程组
$$
\begin{cases}
x = x(t) \quad \alpha \leq t \leq \beta \
y = y(t) \
\end{cases}
$$

或者实自变量的复值函数
$z = z(t) = x(t) + iy(t) \quad \alpha \leq t \leq \beta$
决定的点集$C$称为有向曲线

$A = z(\alpha)$与$B = z(\beta)$分别为$C$的起点和终点

  • 以$B$为起点，$A$为终点的逆此方向的曲线，称为$C$的反向曲线,记作$C^-$

如果当$\alpha \leq t_1 \leq \beta$, $\alpha \leq t_2 \leq \beta$, $t_1 \neq t_2$时，有$z(t_1) \neq z(t_2)$, 那么这时候的$C$为简单有向曲线，或者是简单曲线

当$z(\alpha) = z(\beta)$，为简单闭曲线

当$\alpha \leq t \leq \beta$，$x^\prime(t)$和$y^\prime(t)$均存在、连续，且不全为零，那么$C$是光滑曲线.

  • $C^-:z^- = z^-(t) = z(\alpha + \beta - t)$
  • $z^\prime(\alpha)$指左导数，$z^\prime(\beta)$指右导数
  • 对于光滑闭曲线，必有$z^\prime(\alpha) = z^\prime(\beta)$
